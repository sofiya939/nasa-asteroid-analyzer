# NASA Asteroid Data Analyzer 🚀💻

A Python-based data analytics tool that connects to the open-world **NASA NeoWS (Near Earth Object Web Service) API** to track, parse, and analyze asteroids approaching Earth. 

## 🌐 Geopolitical & Tech Context
In the modern era of Space Policy, satellite security and orbital monitoring are critical parts of global cybersecurity and defense. This project demonstrates how open-source data and automated code can be used to monitor astronomical objects and filter potential global threats, linking aerospace data with digital governance.

## 🛠️ Features
- Connects dynamically to NASA's official live data feeds.
- Calculates average diameters and speed of objects in kilometers per hour.
- Utilizes the `pandas` library for structural data analysis.
- Automatically flags **Potentially Hazardous Objects** to analyze planetary safety.

## 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install requests pandas`
3. Run the script: `python main.py`
